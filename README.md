# 🛒 Global Superstore SQL Analysis

## 📌 Project Overview

This project analyzes the **Global Superstore** dataset using **Oracle SQL** to solve real-world business problems and generate actionable business insights.

The project demonstrates SQL skills ranging from basic data exploration to advanced analytical techniques, including joins, subqueries, Common Table Expressions (CTEs), window functions, views, and indexes.

---

## 🎯 Objectives

- Analyze sales performance across different regions and categories.
- Identify top-performing customers and products.
- Evaluate customer purchasing behavior.
- Analyze product returns and their business impact.
- Demonstrate advanced SQL concepts through practical business scenarios.

---

## 🗂 Dataset

The project uses the **Global Superstore** dataset containing the following tables:

| Table | Description |
|--------|-------------|
| Orders | Customer orders, sales, profit, products, regions, shipping details |
| Returns | Returned orders |
| People | Regional managers assigned to each region |
